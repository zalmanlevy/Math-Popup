Overview

Math Popup is a small always-available notepad. Each line is evaluated as a math expression and the result appears in the right gutter. You can use it like a back-of-the-envelope calculator that keeps a running history, or switch it into plain Text mode and use it as a tray-resident scratchpad.

The window lives in the system tray. Click the tray icon to toggle it, or press Ctrl+Alt+M from anywhere.

Window controls

The header from left to right:

  • Drag handle — click and drag anywhere in the header to move the window.
  • Math / Text — switch the editor between math evaluation and plain text.
  • 📌 Pin — keep the window above other apps. Highlights when active.
  • ? — opens this help page.
  • ⚙ — opens Settings.
  • × — hides the window. The app stays in the tray.

The window is resizable from any edge. Position and size are remembered between sessions.

Math & text modes

Math mode shows two gutters: line numbers on the left (L1, L2, …) and the evaluated result on the right. Syntax highlighting colors numbers, operators, variables, line refs, parentheses, and percent signs.

Text mode hides the result column and turns off math evaluation. Markdown styling (headers, bullets, bold, italic) still renders so you can take notes comfortably.

Numbers & formatting

  • Type any expression: (12 + 5) * 3 → 51.
  • Bare decimals get a leading zero automatically: .5 → 0.5.
  • Numbers with 4+ digits get commas inserted on space, operator, or Enter: 1000 → 1,000. Toggleable in Settings.
  • Commas in input are ignored when evaluating, so 1,000 + 1,000 works.
  • Results are formatted with commas and trimmed to your decimal cap.

Suffixes (k, m, b, t)

Type a number followed by a suffix, then press space — the editor expands it to the full number:

  • 5k → 5,000
  • 1.5m → 1,500,000
  • 2b → 2,000,000,000
  • 3t → 3,000,000,000,000

You can add or edit your own suffixes in Settings → Custom suffixes. Each suffix has a symbol, a multiplier, and a case-sensitivity flag.

If you turn off "Expand suffixes in editor", the typed text stays as 5k but the evaluator still computes the right value.

Variables

Define a variable on its own line, then reference it later:

price = 12.50
qty   = 7
price * qty

Variable names use letters, digits, and underscores, and must start with a letter or underscore. Reserved names like L1, pi, e, true, and false cannot be redefined.

Line references

Refer to the result of any earlier line with L followed by the line number. Case doesn't matter — L1 and l1 both work:

1,000 + 250
L1 * 0.07

Line refs always pull the most recent successful value of that line. If a referenced line doesn't have a number, the reference resolves to NaN.

Percent & basis points

% after a number means "of one": 50 * 20% → 10. With a leading + or - it becomes "relative to the left side":

  • 200 + 10% → 220 (10% of 200, added)
  • 200 - 5% → 190

bps (or bp) is basis points — one hundredth of a percent. Same relative behavior:

  • 1,000,000 + 25bps → 1,002,500
  • 50bps → 0.005

Functions & constants

Powered by mathjs, so you have access to a large standard library. Common built-ins recognized by the syntax highlighter:

  • Constants: pi, e, tau, phi
  • Trig: sin, cos, tan, asin, acos, atan, atan2
  • Logs & exp: log, log2, log10, ln, exp
  • Roots & rounding: sqrt, abs, round, floor, ceil
  • Aggregates: min, max, sum, mean, median, mod

Examples: sqrt(2), sin(pi / 4), round(123.456, 1).

Markdown

Light markdown styling renders inline:

  • # Heading, ## Heading, ### Heading — header lines are not evaluated.
  • - item or * item — bulleted line. The content after the bullet is still evaluated as math (in math mode).
  • **bold** and *italic* — inline emphasis.

Tray & startup

  • Click the tray icon to toggle the popup. Double-click works too.
  • Right-click the tray icon for a menu: Open, Settings, Quit.
  • Closing the window only hides it — the app stays in the tray. Use the tray menu's "Quit" to exit.
  • "Launch when Windows starts" puts the app in your login items. It starts hidden, so only the tray icon appears until you click it.

Settings

  • Theme — System (follows your OS), Light, or Dark. System mode updates automatically when you change your OS color scheme.
  • Launch when Windows starts — auto-start hidden in the tray.
  • Auto-format numbers with commas — adds thousands separators as you type.
  • Expand suffixes in editor — turns 5m into 5,000,000 on space. Off keeps the typed form.
  • Max decimal places — caps how many decimals are shown in results. Integers omit the decimal point entirely.
  • Custom suffixes — add your own multipliers (e.g. x = 1000, kg = 1, etc.). Each row has a symbol, multiplier, and case-sensitive toggle.

Storage & privacy

Everything is stored locally on your machine. Your note text, window position, and settings are saved to a settings.json file in the app's user-data folder. Nothing is sent over the network.

Your note auto-saves a few hundred milliseconds after you stop typing.
